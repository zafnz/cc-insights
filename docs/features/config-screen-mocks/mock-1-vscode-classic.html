Settings

Appearance
Appearance: Agents List Location
Controls where the agents panel is displayed. Use Stacked to show both sessions and agents in the sidebar.
Appearance: Bash Tool Summary
How to display bash commands in the conversation. Description shows what the command does, Command shows the actual command text.
Appearance: Relative File Paths
Show file paths relative to the project directory instead of absolute paths. Makes tool output more readable.
Appearance: Show Timestamps
Display timestamps in the conversation view. Helps track when messages were sent during long sessions.
Appearance: Timestamp Idle Threshold
Only show timestamps after this many minutes of inactivity. Set to 0 to show timestamps on every message.
Behavior
Behavior: Auto-scroll on Message
Automatically scroll to the bottom when new messages arrive. Disable to stay at your current scroll position while reviewing output.
Behavior: Notify on Prompt
Show a system notification when Claude asks for input. Useful when running long tasks in the background.
Session Defaults
Session: Default Model
The model to use for new sessions. Sonnet is faster and cheaper. Opus is more capable for complex tasks.
Developer
Developer: Debug Logging
Write debug information to the console. Enable when troubleshooting issues with the app.
Developer: Show Raw Messages
Show a debug button on messages to inspect the raw JSON from the SDK. Useful for understanding the protocol.
