Config Screen Mockups

4 design approaches for a desktop-native settings experience

1. VSCode Classic
Faithful to VSCode's settings layout. Category:Name format, inline dropdowns, toggles with descriptions on the left. Search bar at top. Maximum information density.
Familiar
2. Compact Cards
Card-based sections with compact rows. Controls on the right, descriptions truncated to single line. Modified indicator bar. Very scannable.
Clean
3. Sidebar Navigation
Categories in left sidebar, settings in main area. More room for descriptions. Good for many categories. System Preferences style.
Spacious
4. Minimal Grid
Collapsible sections with pill selectors for enums. Centered layout with gradient header. Modern, playful feel. Interactive collapse.
Modern
