Appearance

Customize how CC Insights looks and displays information

Agents List Location
Controls where the agents panel is displayed. Vertical Panel shows agents in a dedicated sidebar column. Stacked shares the sidebar between sessions and agents. Horizontal Toolbar shows agents above the conversation.
Bash Tool Summary
How to display bash commands in the conversation. Description shows a human-readable summary of what the command does. Command shows the actual command text.
Relative File Paths
Show file paths relative to the project directory instead of absolute paths. Makes tool output more compact and readable when working within a project.
Show Timestamps
Display timestamps next to messages in the conversation view. Helps track timing during long debugging sessions.
Timestamp Idle Threshold
Only show timestamps after this many minutes of inactivity. Set to 0 to show timestamps on every message regardless of timing.
